### 1. ArXiv 2605.04004 — Structural Limits of OHLCV-Based Intraday Signals (FALSIFICATION)
**URL:** https://arxiv.org/abs/2605.04004  
**Date:** 2026-05-05 | **Category:** q-fin.TR

**Finding:** A systematic falsification study of 14 intraday momentum signal families on MNQ futures (947 trading days, 5-min data, 2021-2025). ALL signals failed strict institutional criteria: out-of-sample walk-forward validation, T-stat ≥ 2.0, ≥30 trades, positive net return after 2-point round-trip cost, multi-year stability.

- Gross edge: 0.07–1.50 points per trade — insufficient to overcome transaction costs.
- Best performer (gap continuation): T = 3.23, +14.52 points, but N = 22 (fails sample size).
- Two validated signals from a separate research program included as positive controls → methodology CAN detect genuine edge when present.

**Implication:** Simple OHLCV-derived intraday strategies are structurally unprofitable under realistic execution. This includes opening range breakouts, gap strategies, volume signals, cross-session momentum, liquidity grabs, volatility-conditioned classifiers. Do NOT waste development cycles here.

**Source confidence:** ⭐⭐⭐⭐⭐ (reproducible falsification framework, positive controls, 5-year data)

| DJI30 | $146K | 10–18% | 3.3–4.8 | 2.55–3.6 |
| EUR/USD | $69K | 10–18% | 3.3–4.8 | 2.55–3.6 |
| XAU/USD | $125K | 10–18% | 3.3–4.8 | 2.55–3.6 |
| Total | $340K | — | ~676/day avg | 2.55–3.6 |

**Methodology:** 30-day optimization window + 7-day test window. Strategy uses VGRSI crossover signals.

**Implication:** This is a genuinely novel technical indicator with published profitability. Action: implement VGRSI in Python, backtest on ccxt data with walk-forward validation. Paper does not disclose full algorithm — need to reconstruct from visibility graph literature (Lacasa et al. 2008).

## 📝 METHODOLOGY NOTES

### What Worked
1. **GitHub search API + save-to-disk + parse** — circumvented Tirith curl-pipe blocks successfully.
2. **ArXiv API via HTTPS** — zero auth, zero rate limits, rich structured data. Best source for quant papers.
3. **Hacker News Algolia API** — fast, no auth, good signal for trending agentic/AI topics.
4. **TechCabal RSS** — active, dated, relevant for SA market intelligence.
5. **Deep-read raw READMEs and architecture specs** — AlphaTeam ARCHITECTURE.md and lyrie Shield Doctrine were the highest-signal documents of the session.

### What Failed
1. **Google search via browser** — blocked immediately from cloud IP (CAPTCHA/sorry page).
2. **DuckDuckGo HTML search** — returned empty/malformed HTML. Not reliable for programmatic extraction.
3. **GitHub search API rate limit** — unauthenticated = 10 requests/minute. We hit it after 3-4 searches. Raw content URLs (raw.githubusercontent.com) still work fine.
4. **HN search with `created_at_i` numeric filter** — returned empty JSON. Without the filter, results are valid but not time-bounded.
